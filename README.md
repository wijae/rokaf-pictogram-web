# ROKAF pictogram web

## Project info
본 사이트는 대한민국 공군에서 배포하는 공군 픽토그램을 쉽고 간편하게 찾을 수 있도록 만들어졌습니다.

공군 픽토그램의 저작권은 대한민국 공군에 있으며, 출처 표기 시 누구나 자유롭게 사용할 수 있습니다. 픽토그램 원본 파일은 공군 공감블로그 (https://afplay.kr/2318)에서 내려받을 수 있습니다
